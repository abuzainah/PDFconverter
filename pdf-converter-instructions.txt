================================================================================
PART 1: PROJECT OVERVIEW
================================================================================

Project Name: PDF Quick Convert

Purpose: A free online tool that converts PDF files to PNG, JPG, TXT, and DOCX 
formats. All processing happens in the browser for privacy.

Technology Stack:
- Next.js 14 with App Router
- TypeScript
- Tailwind CSS
- Client-side PDF processing

Deployment Target: Cloudflare Pages (static export)

Monetization: Google AdSense advertisements

Key Requirements:
- Must be fully static (no server-side processing)
- Must work entirely in the browser
- Must comply with Google AdSense policies
- Must be mobile responsive
- Must have all required legal pages


================================================================================
PART 2: DEPENDENCIES TO INSTALL
================================================================================

After creating the Next.js project, install these packages:

Production dependencies:
- react-dropzone (for file upload drag and drop)

PART 22: CONVERSION LIBRARY SPECIFICATIONS
================================================================================

FILE: lib/conversion/pdfToImages.ts

PURPOSE:
Convert PDF pages to PNG or JPG images

FUNCTIONALITY:
- Accept a PDF file, format choice (png/jpg), and progress callback
- Load PDF using pdfjs-dist library
- Configure PDF.js worker from CDN for Cloudflare compatibility
- Loop through each page of the PDF
- Render each page to a canvas element
- Convert canvas to image blob (PNG or JPG)
- For JPG, use quality setting of 0.92
- Use scale of 2 for high quality output
- Call progress callback after each page
- Return array of converted files with name, blob, and type

ERROR HANDLING:
- Handle corrupted PDFs
- Handle password-protected PDFs
- Handle canvas rendering failures


FILE: lib/conversion/pdfToText.ts

PURPOSE:
Extract text content from PDF

FUNCTIONALITY:
- Accept a PDF file and progress callback
- Load PDF using pdfjs-dist library
- Loop through each page
- Extract text content from each page
- Combine text with page separators
- Call progress callback after each page
